## What is QMSAuditor?

QMSAuditor is an open-source audit
management platform designed for
Quality Managers in SMEs who need
enterprise-grade audit tools —
without the enterprise price tag.

Enterprise audit software costs
$15,000–$150,000 per year.

Most Quality Managers don't work
for companies that can afford that.

This is the alternative.

---

## Features (MVP — In Progress)

- 🔄 User authentication & roles
- 🔄 Audit plan creation & scheduling
- 🔄 ISO 9001 clause-mapped checklists
- 🔄 Finding & nonconformance logging
- 🔄 Corrective action tracking
- 🔄 Dashboard & reporting
- ⏳ Evidence attachment (MVP+)
- ⏳ PDF report export (MVP+)
- ⏳ Email notifications (MVP+)

## Why Open Source?

Because quality management
should not be a luxury.

ISO 9001 is a global standard.
The tools to implement it
should be accessible to every
organisation — regardless of size
or budget.

QMSAuditor is built by a
Professional Engineer and Quality
Manager who lives this problem daily.
